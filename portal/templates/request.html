Virtual Cluster States

  • New

    The virtual cluster has been defined, but has not yet been processed in any way.

  • Initializing

    The virtual cluster has been validated, and a new headnode is being setup.

  • Running

    The factory has been configured, and at least some of the workers have started to run.

  • Terminate

    A terminate action for the virtual cluster has been received, and workers start to be vacated.

  • Cleanup

    The headnode is shutdown, other minor bookeping of the virtual cluster is deleted.

  • Terminated

    The lifetime of the virtual cluster comes to an end. The cluster remains in this stage until it is explicitly deleted.
